# Taxi_Trip_Analysis_PowerBI

Overview

This project presents an interactive Power BI report built using taxi trip data. The report helps analyze trip performance, revenue, passenger trends, and payment methods through interactive visualizations.

# Objectives

Analyze taxi trip data using Power BI.
Visualize key performance indicators (KPIs).
Identify trends in trips, revenue, and passenger behavior.
Enable interactive data exploration using filters and slicers.

# Tools Used

Power BI
Microsoft Excel
Report Features
KPI Cards (Total Trips, Revenue, Passenger Count, Average Trip Distance)
Line Charts for trip trends
Bar Charts for location and payment analysis
Interactive Slicers and Filters
Multiple report pages for detailed analysis

# Key Insights

Identified peak periods with the highest trip volume.
Compared payment methods used by passengers.
Analyzed trip distribution across different locations.
Monitored revenue and average trip distance through interactive reports.
